# Iterate_names
CMD tool for building new names from sentence

How to use
```
usage: main.py [-h] [-v] [-i INPUT] [-o OUTPUT]

CMD tool that accept file path and iterate lines, each line split by space and build new lines with all words in all orders.

options:
  -h, --help            show this help message and exit
  -v, --version         show program's version number and exit
  -i INPUT, --input INPUT
                        Input file path
  -o OUTPUT, --output OUTPUT
                        Output file path```
